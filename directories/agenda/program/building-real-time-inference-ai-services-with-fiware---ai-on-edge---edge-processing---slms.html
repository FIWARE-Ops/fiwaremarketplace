An IoT Agent is an implementation of the Backend Device Management Generic Enabler, according to the FIWARE reference architecture. IoT Agents translate IoT-specific protocols into the NGSI context information protocol, that is the FIWARE standard data exchange model. You use an IoT Agent because you need to connect objects to gather data or interact with them, typical IoT use case scenario. Therefor you need to use or extend one of the existing IoT Agents that are part of the FIWARE Catalogue.
