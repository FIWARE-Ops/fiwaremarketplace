Digital Enabler TM is a versatile tool designed to integrate and harmonize diverse data sources, including web-based data, IoT device outputs, open data portals, and legacy systems. It ensures seamless communication between these systems by evaluating metadata quality and preventing data duplication. The platform transforms these data sources into accessible, meaningful information through a user-friendly interface and standard APIs.

With its innovative graphical and map-based tools, Digital Enabler empowers decision-makers and developers to design data correlations and create interactive dashboards. These dashboards facilitate real-time data analysis and visualization, enabling timely decision-making. For example, it can harmonize real-time data to optimize public transport supply.

Digital Enabler allows the organization of dashboards and applications into specific domains such as environment, mobility, or manufacturing. This feature is particularly beneficial for businesses and developers, as it enables the creation of new, reusable services and applications that are portable and replicable across different contexts. The platform eliminates vendor lock-in and offers several key functionalities:

  • Automatic Discovery of Data Sources: Identifies data sources from the web and assists in creating a "Single Point of Knowledge (SPOK)."
  • Data Integration and Harmonization: Facilitates graphical integration of data from various sources and adapts any data structure to NGSI LD FIWARE Data Models.
  • Automatic Information Deduction: Utilizes a repository of machine learning models and algorithms to derive new insights, with tools available for data scientists.
  • Data Analysis and Visualization: Supports the creation of user-friendly dashboards and mobile apps, leveraging data from the SPOK for reuse across different customers.

Digital Enabler is fully compliant with EU GDPR regulations, ensuring proper management of consents and personal data. It adheres to OASC principles, offering open access through standard APIs and ensuring portability and replicability. The platform meets all three compliance levels of the Minimal Interoperability Mechanisms (MIMs).
